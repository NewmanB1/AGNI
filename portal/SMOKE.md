# Portal manual smoke (5–10 min)

Run `npm run portal` (or any static server on `portal/`), open `#/settings`.

1. **Settings → Test hub key** — Set Hub URL + key → **Test hub key** → expect toast “accepted” or clear reject message.
2. **Author** — Log in → `#/author` → sticky bar on edit flow; **Jump to step**; collapse step ▼/▶.
3. **Browse** — `#/author/browse` → **Sort** dropdown changes order without new search; **Preview** opens tab.
4. **Collab** — `#/hub/collab` → polling line; **Deny** → toast + live region.
5. **Students (non-admin)** — With a non-admin key, `#/students` → admin/permission copy, link to Learn/Groups as needed.

Stub/i18n: `#/hub`, `#/parent/dashboard`, `#/admin/onboarding` — copy follows language in Settings.
